# 医院后台管理系统 java904

## 项目概述

医院后台管理系统是基于 SpringBoot 与 Vue 的前后端分离架构，旨在提供一套功能完善、易于扩展的管理系统，满足医院日常运营管理的需求。

## 技术栈

### 前端技术
- HTML
- CSS
- JavaScript
- Vue

### 后端技术
- SpringBoot
- Mybatis

### 开发环境
- Eclipse/IDEA

### 数据库
- MySQL 5.x/8

### JDK
- 8.0

## 功能模块

系统包含但不限于以下功能模块：
- 用户管理
- 权限控制
- 病历管理
- 药品管理
- 医生排班
- 患者预约

## 系统角色

系统定义了以下角色：
- 系统管理员
- 医生
- 药剂师
- 患者服务人员

## 运行环境

- Java 8 或更高版本
- MySQL 5.x 或 8.x
- 支持主流浏览器

## 部署步骤

1. 准备 Java 运行环境与 MySQL 数据库。
2. 导入数据库脚本。
3. 部署后端服务。
4. 部署前端应用。

## 目录结构

```
project
├── backend                # 后端代码
├── frontend               # 前端代码
└── db                     # 数据库脚本
```

## 核心亮点

- 前后端分离，便于维护与扩展。
- 基于角色的权限管理，保障系统安全。
- 高度模块化，易于定制开发。
